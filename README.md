# simple-paginate
Pagination in a simple way

# Install
- Visual Studio
- git clone https://github.com/rodrigogaraujo/simple-paginate.git
- run in the terminal: yarn
- run in the terminal: yarn start


# Libraries & Features
- React
- useState
- useEffect
- axios
- styled-components
